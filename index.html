Konsep Inti Lua (Lanjutan)

Selami lebih dalam fitur-fitur yang membuat Lua kuat. Setiap topik mencakup **pengertian mendalam** dan **contoh kode praktis**.

Metatable: Memperluas Perilaku Table

**Metatable** adalah kunci untuk implementasi **OOP** dan *operator overloading* di Lua. Ia memungkinkan Anda mengubah cara table berinteraksi, misalnya, mendefinisikan apa yang terjadi saat dua table dijumlahkan (`__add`).

Contoh Code (Ringkas): Overloading

local mt = {
    __add = function(a, b) return a.value + b.value end
}
local obj1 = {value = 5}
local obj2 = {value = 10}

setmetatable(obj1, mt)
print(obj1 + obj2) -- Output: 15

Coroutines: Eksekusi yang Kooperatif

Coroutines memungkinkan **pause dan resume** fungsi. Ini menyediakan *multitasking* kooperatif yang sangat efisien dan ringan, ideal untuk *game loop* atau alur kerja I/O asinkron.

Table Hybrid: Array, Map, dan Object dalam Satu

Table adalah satu-satunya struktur data kompleks di Lua. Ia dapat berfungsi sebagai *Array* (indeks numerik) dan *Hash Map* (indeks string) secara bersamaan, memberikan fleksibilitas luar biasa.

Pertanyaan yang Sering Diajukan

Keunggulan utama Lua adalah **kecepatannya**, **ukuran *footprint*-nya yang sangat kecil** (sekitar 250KB), dan **kemampuan integrasinya** yang superior dengan C/C++ (sebagai *embedded language*). Ini ideal untuk *scripting* game dan sistem tertanam.

Lua sangat populer di **Game Development** (Roblox, World of Warcraft Addons, CryEngine) dan **Network/System Scripting** (Nginx, Redis, Wireshark). Kecepatannya di lingkungan *high-performance* tidak tertandingi.
