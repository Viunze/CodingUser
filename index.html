Metatable: Memperluas Perilaku Table

**Metatable** adalah jantung dari fleksibilitas Lua. Ini memungkinkan kita mengubah perilaku table (seperti operator aritmatika, akses indeks, atau bahkan implementasi *inheritance*). Metatable bekerja dengan *metamethods* seperti `__index` dan `__add`. Fitur ini sangat penting untuk implementasi **Object-Oriented Programming (OOP)** di Lua.

Contoh Code: Penggunaan `__add`

local Vektor = {}
local metatableVektor = {
    __add = function(v1, v2)
        return Vektor.new(v1.x + v2.x, v1.y + v2.y)
    end
}

function Vektor.new(x, y)
    local self = {x = x, y = y}
    setmetatable(self, metatableVektor)
    return self
end

local vA = Vektor.new(10, 5)
local vB = Vektor.new(3, 2)

local vC = vA + vB  -- Ini memanggil fungsi __add
print(vC.x)          -- Output: 13

Coroutines: Eksekusi yang Kooperatif

**Coroutines** (sering disebut *thread* ringan) adalah fitur untuk menjalankan fungsi secara kooperatif. Berbeda dengan *thread* paralel, Coroutines **menyerahkan kontrol** kembali ke program utama, memungkinkan *state* program dibekukan dan dilanjutkan. Ini ideal untuk manajemen animasi game, I/O non-blokir, atau alur kerja yang kompleks.

Contoh Code: Menggunakan `yield` dan `resume`

local function hitunganMundur(n)
    while n > 0 do
        print("Sisa waktu:", n)
        n = n - 1
        coroutine.yield() -- Serahkan kontrol
    end
    return "Selesai!"
end

local co = coroutine.create(hitunganMundur)

-- Memanggil resume setiap frame/loop
print(coroutine.resume(co, 5)) -- Sisa waktu: 5
print(coroutine.resume(co))    -- Sisa waktu: 4
print(coroutine.resume(co))    -- Output: true, Selesai!

Tipe Data & Table: Table Adalah Segalanya

Lua dikenal karena kesederhanaan tipe datanya (hanya delapan). Yang paling dominan adalah **table**, yang berfungsi ganda sebagai *array*, *hash map*, *object*, dan *struct*. Fleksibilitas ini meminimalkan kebutuhan akan struktur data terpisah dan membuat kodenya ringkas.

Contoh Code: Fleksibilitas Table

-- Table sebagai Array
local arr = {"a", "b", "c"}
print(arr[2]) -- Output: b

-- Table sebagai Map/Object
local player = {
    nama = "Kaito", 
    level = 10, 
    ["kekuatan-khusus"] = 50
}
print(player.nama)
print(player["kekuatan-khusus"])

Pertanyaan yang Sering Diajukan

Keunggulan utama Lua adalah **kecepatannya**, **ukuran *footprint*-nya yang sangat kecil** (sekitar 250KB), dan **kemampuan integrasinya** yang superior dengan C/C++ (sebagai *embedded language*). Ini membuatnya ideal untuk *scripting* game dan sistem tertanam, di mana performa dan sumber daya sangat dibatasi.

Lua sangat populer di **Game Development** (Roblox, World of Warcraft Addons, CryEngine) dan **Network/System Scripting** (Nginx, Redis, Wireshark). Kecepatannya di lingkungan *high-performance* tidak tertandingi oleh bahasa *scripting* lain.
